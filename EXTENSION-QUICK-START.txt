# OpenPilot Extension - Development Mode Guide

## Quick Start (Works Immediately!)

1. Open VS Code
2. Press: Ctrl+Shift+P
3. Type: 'Open Folder'
4. Select: i:\openpilot\vscode-extension
5. Press F5 (or Run > Start Debugging)

This opens a new VS Code window with the extension loaded!

## Using the Extension:

In the new window:
- Press Ctrl+Shift+P  'OpenPilot: Open Chat'
- Select code  Ctrl+Shift+P  'OpenPilot: Explain Code'
- All commands work!

## Why This Works:

Development mode doesn't need packaging - it runs directly from source
with all dependencies available.
